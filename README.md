# Youtube2View

Watch Youtube and chat with friends, all in one app!

## Dependencies

* Node.JS
* HTML/CSS/JS
* Bootstrap 4
* socket.io

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
or
